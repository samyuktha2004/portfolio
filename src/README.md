# 🌸 The Bubbly Detective Portfolio

Interactive portfolio with kawaii-inspired detective theme. Built with React, TypeScript, and Tailwind CSS.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## 📁 Structure

```
/
├── components/          # UI components
│   ├── book-sections/  # Portfolio book pages
│   ├── tablet-sections/# Pink tablet modals
│   └── ui/             # Reusable components
├── constants/          # Theme values (colors, spacing, etc.)
├── data/               # Portfolio content (EDIT HERE)
├── hooks/              # Custom React hooks
├── utils/              # Helper functions
└── docs/               # Documentation
```

## ✨ Features

- 🏠 **Detective Room** - Interactive landing with clickable hotspots
- 📱 **Pink Tablet Modals** - Detailed content overlays
- 📖 **Portfolio Book** - Traditional scrollable view
- 🔄 **Flippable Avatar** - 3D flip animation
- 📄 **Resume Download** - Print-ready PDF generation

## 🎨 Editing Content

All content lives in `/data/portfolioData.ts`. Edit once, updates everywhere.

See [Content Guide](/docs/CONTENT.md) for details.

## 🛠️ Development

- **Theme constants**: `/constants/theme.ts`
- **Custom hooks**: `/hooks/`
- **Reusable UI**: `/components/ui/`

See [Developer Guide](/docs/DEVELOPMENT.md) for details.
